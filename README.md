Aleph — 这是这个仓库的名字，它是博尔赫斯短篇小说中一个点，能同时看见宇宙所有事物。基于Obsidian提供的能力以及[[卢曼笔记法]]的思想，知识库不再是单一树状结构，而是变成可以自下向上生长的图谱。

这是一个基于[[PARA]] + 卢曼笔记法，以Obsidian为载体搭建起来的个人知识库，旨在通过构建第二大脑，实现知识资产的累积效应，达到知识复利的目的。

- 00 Inbox（收件箱）：所有新想法、网页剪报的默认存放地，用于无摩擦的快速捕获。
- 01 Projects（项目）：存当前正在进行、有明确截止日期的任务。
- 02 Areas（领域）：存放需要长期关注和维持的标准。
- 03 Resources（资源）：存放“他人的思想”，比如你阅读书籍、文章后做的文献笔记和摘要。
- 04 Archive（归档）：存放已完成的项目和不再活跃的资料，作为冷存储。
- 10 Zettelkasten（卡片盒）：最核心的知识库。这里存放经过深度思考、用自己话写成的“永久笔记（Permanent Notes），也就是自己的思想资产。
- 20 复盘：每周复盘


具体操作步骤：
1. 快速捕获，存入Inbox。
2. 提炼外部知识，存入资源（文献笔记）。
3. 用费曼技巧，总结资源提炼观点，存入卡片盒。
4. 在项目中使用知识，通过链接方式引用  知识。
